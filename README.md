# Holbox Dining · Grupo Mittoz

Landing page de reservaciones para los 4 negocios de Grupo Mittoz en Isla Holbox:
- El Nido
- Soulbox (rooftop)
- Bocamar
- Leonessa

## Programa de referidos

Las reservas se generan exclusivamente con un código de referidor (recepcionistas de hoteles aliados). Cada reserva genera comisión del 5% sobre el subtotal antes de IVA cuando el cliente asiste y se captura el consumo.

## Stack

- React 18 + Vite
- Supabase (base de datos y API REST)
- Lucide React (iconos)
- Cormorant Garamond + Italiana + Inter (tipografías de Google Fonts)

## Desarrollo local

```bash
npm install
npm run dev
```

## Deploy

Configurado para deploy automático en Vercel desde la rama `main`.
